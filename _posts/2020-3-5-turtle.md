```
import turtle 
wn = turtle.Screen() # background
bonny = turtle.Turtle() # class under a module
```

Colors
```
bonny.color("red")
```

https://trinket.io/docs/colors

```
bonny.right(72) # angle
bonny.forward(50) # distance
```

```
bonny.circle(100, 180) # semicircle
```

Pen up or down
```
bonny.penup()
bonny.pendown()
```
